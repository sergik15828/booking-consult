# Booking Consult

WordPress-плагин для онлайн-записи на консультации. Клиенты выбирают услугу, дату и свободный слот прямо на сайте, а администратор управляет расписанием через панель WordPress.

## Возможности

**Для клиентов:**
- Выбор услуги из списка
- Календарь на 30 дней вперёд с отметками доступных дат
- Просмотр свободных временных слотов
- Бронирование с указанием имени, email, телефона и комментария

**Для администратора:**
- Управление услугами (название, длительность, пресеты времени)
- Настройка доступности — какие окна открыты на каждую дату
- Просмотр и удаление записей

## Стек

- PHP 7.4+, WordPress 5.0+
- MySQL / MariaDB (через `$wpdb`)
- Vanilla JS + Fetch API
- WordPress REST API

Внешних зависимостей нет.

## Установка

1. Скопируйте папку `booking-consult` в `wp-content/plugins/`
2. Активируйте плагин в разделе **Плагины** админ-панели WordPress
3. При активации автоматически создаются таблицы и три демо-услуги

## Использование

### Шорткод

Вставьте на любую страницу или запись:

```
[booking_consult]
```

Отобразится виджет бронирования: выбор услуги → календарь → слоты → форма записи.

### Админ-панель

В боковом меню WordPress появится раздел **Booking Consult** с тремя страницами:

- **Услуги** — редактирование названий, длительности и пресетов времени
- **Записи** — список всех бронирований
- **Доступность** — визуальный календарь для настройки открытых окон по каждой услуге

## REST API

Базовый путь: `/wp-json/bc/v1/`

### Публичные

| Метод | Путь | Параметры | Описание |
|-------|------|-----------|----------|
| GET | `/services` | — | Список активных услуг |
| GET | `/calendar` | `service_id` | Календарь на 30 дней с флагами доступности |
| GET | `/slots` | `service_id`, `date` (Y-m-d) | Свободные слоты на дату |

### Требуют авторизации

| Метод | Путь | Описание |
|-------|------|----------|
| POST | `/book` | Создание записи (нужен логин + nonce) |

### Админские (capability: `manage_options`)

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/admin/availability` | Окна доступности для услуги |
| POST | `/admin/availability` | Сохранение окон на дату |

## База данных

Плагин создаёт три таблицы (с префиксом WordPress):

| Таблица | Назначение |
|---------|------------|
| `wp_bc_services` | Услуги (название, длительность, шаг слотов, пресеты) |
| `wp_bc_appointments` | Записи клиентов (контакты, время, статус) |
| `wp_bc_availability` | Окна доступности (услуга + дата + время от/до) |

Бронирование использует MySQL-транзакции для защиты от двойных записей на один слот.
